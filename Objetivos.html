OBJETIVOS

2.1 OBJETIVO GENERAL

Desarrollar una aplicación didáctica que enseñe los aspectos relevantes que se implementan en la media técnica, por otra parte, también permite reforzar algunas competencias como lo es el pensamiento computacional.

2.2 OBJETIVOS ESPECÍFICOS

  • 2.2.1 Definir las necesidades del cliente para identificar el problema y buscarle una solución al mismo.
  • 2.2.2 Analizar las necesidades del cliente para alcanzar la meta de forma satisfactoria.
  • 2.2.3 Diseñar prototipos que contribuyan a la buena estética y buen desarrollo de la herramienta.
  • 2.2.4 Codificar la aplicación para obtener una herramienta funcional y actualizable.
  • 2.2.5 Probar los códigos utilizados para verificar que no haya error.
  • 2.2.6 Validar la herramienta con ayuda del cliente para descartar cualquier tipo de anomalía en la misma.
  • 2.2.7 Actualizar la herramienta, dando nuevas características y darles solución a posibles errores.
